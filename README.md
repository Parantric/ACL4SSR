## ✍️ **常用正则**

```javascript
(A).*(B)        节点名既有 A又有 B
(A)|(B)         节点名有 A 或者 B  
^((?!A).)*$     节点名不含有 A
(?!.*(A)).*(B)  节点名不含有 A，同时含有 B
```
